# url-shortener
<h2>instructions</h2>
<ol>
  <li>
    <h3>run project</h3>
    <ol>
      <li>
        <p>setup dependencies</p>
        <code>make setup</code>
      </li>
      <li>
        <p>build application</p>
        <code>go build -o url-shortener-app app/*.go</code>
      </li>
      <li>
        <p>run application</p>
        <code>./url-shortener-app</code>
      </li>
      <li>
        <p>tear-down dependencies</p>
        <code>make tear-down</code>
      </li>
    </ol>
  </li>
  <li>
    <p>run test</p>
    <code>go test ./...</code>
  </li>
</ol>
